botprofile_blacklist_binding

Binding object showing the blacklist that can be bound to botprofile.

Read/write properties

bot_blacklist_action<String[]>

One or more actions to be taken if bot is detected based on this Blacklist binding. Only LOG action can be combined with DROP or RESET action.
Default value: NONE
Possible values = NONE, LOG, DROP, RESET, REDIRECT

bot_blacklist_enabled<String>

Enabled or disbaled black-list binding.
Default value: OFF
Possible values = ON, OFF

bot_bind_comment<String>

Any comments about this binding.
Minimum length = 1

bot_blacklist_type<String>

Type of the black-list entry.
Possible values = IPv4, SUBNET, IPv6, IPv6_SUBNET, EXPRESSION

name<String>

Name for the profile. Must begin with a letter, number, or the underscore character (_), and must contain only letters, numbers, and the hyphen (-), period (.), pound (#), space ( ), at (@), equals (=), colon (:), and underscore (_) characters. Cannot be changed after the profile is added. The following requirement applies only to the Citrix ADC CLI: If the name includes one or more spaces, enclose the name in double or single quotation marks (for example, "my profile" or 'my profile').
Minimum length = 1
Maximum length = 31

bot_blacklist_value<String>

Value of the bot black-list entry.

bot_blacklist<Boolean>

Blacklist binding. Maximum 32 bindings can be configured per profile for Blacklist detection.

logmessage<String>

Message to be logged for this binding.
Minimum length = 1

__count<Double>

count parameter
