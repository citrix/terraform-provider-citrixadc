nslicenseparameters

Configuration for licenseparameters resource.

Read/write properties

alert1gracetimeout<Double>

If ADC remains in grace for the configured hours then first grace alert will be raised.
Default value: 6
Minimum value = 0
Maximum value = 24

alert2gracetimeout<Double>

If ADC remains in grace for the configured hours then major grace alert will be raised.
Default value: 240
Minimum value = 24
Maximum value = 720

licenseexpiryalerttime<Double>

If ADC license contract expiry date is nearer then GUI/SNMP license expiry alert will be raised.
Default value: 30
Minimum value = 1
Maximum value = 365

heartbeatinterval<Double>

Heartbeat between ADC and Licenseserver is configurable and applicable in case of pooled licensing.
Default value: 280
Minimum value = 30
Maximum value = 280

inventoryrefreshinterval<Double>

Inventory refresh interval between ADC and Licenseserver is configurable and applicable in case of pooled licensing.
Default value: 360
Minimum value = 45
Maximum value = 1440

Read only properties

_nextgenapiresource<String>

.

Operations

You can perform the following operations:

updateunsetget (all)

Some options that you can use for each operation:

  • Getting warnings in response: NITRO allows you to get warnings in an operation by specifying the "warning" query parameter as "yes". For example, to get warnings while connecting to the NetScaler appliance, the URL is as follows:
    http://<netscaler-ip-address>/nitro/v1/config/lbvserver?warning=yes

    If any, the warnings are displayed in the "X-NITRO-WARNING" header and the HTTP status code is 209.

  • Authenticated access for individual NITRO operations: One of the authentication mechanisms is OneTimelogin/Basic authentication. You can use the below two headers specifying username and password and send it along with the regular nitro request.

    X-NITRO-USER:<username>
    X-NITRO-PASS:<password>

    Note: In such cases, make sure that the request header DOES not include the following:
    Cookie:NITRO_AUTH_TOKEN=<tokenvalue>
